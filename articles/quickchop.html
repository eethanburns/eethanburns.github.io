QuickChop for Chrome

Release Date TBD

A Chrome extension for rapidly sampling sounds from the web, to replace YouTube to MP3 services.

-Eliminate the need for clunky and unreliable ‘YouTube to MP3’ services.

-Use keyboard input from user to slice audio and experiment immediately with sounds.

-Make organization of downloaded sounds seamless and convenient.

-User Interface window with visual feedback and interactive options

Chrome Web Store release coming June 2025
